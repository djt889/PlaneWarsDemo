# 飞机大战demo，制作中。。。
基于单例模式开发飞机大战Demo，包含常规移动和视角切换功能，提升用户体验。</br>
实现账号注册和登录功能，确保用户信息安全和个性化体验。</br>
提供可切换武器、改变子弹形态和频率的UI，增加游戏乐趣。</br>
支持切换飞机功能，各飞机具有不同速度和攻击力，丰富游戏玩法。</br>
包含死亡坠落和爆炸动画效果，提升游戏真实感。</br>
敌人具备射线检测和避障功能，增加游戏挑战性和趣味性。</br>
